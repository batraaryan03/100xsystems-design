Welcome to Lumina

Lumina is a design collaboration platform that helps teams build, share, and maintain design systems at scale. This documentation covers everything from getting started to advanced API usage.

Overview

Lumina bridges the gap between design and engineering. With real-time collaboration, version-controlled design systems, and automated code generation, teams can ship beautiful products faster than ever.

💡

New to Lumina? Check out the Quickstart Guide to get up and running in under 5 minutes.

Quickstart

Install the Lumina CLI and initialize your first project:

npm install -g lumina-cli
lumina init my-project
cd my-project
lumina dev

This creates a new project with the default design system template. Open http://localhost:3000 to see your project in action.

Core Features

Design Tokens

Define your design primitives — colors, typography, spacing, and shadows — as structured tokens. Tokens can be exported to CSS, JSON, or Tailwind config.

{
  "color": {
    "primary": { "500": "#6366f1" },
    "neutral": { "100": "#f1f5f9", "900": "#0f172a" }
  },
  "spacing": { "sm": "8px", "md": "16px", "lg": "24px" }
}

Component Library

Build and document reusable components with interactive previews, code snippets, and version history. Each component can have multiple variants and states.

✨

Pro tip: Use lumina generate button to scaffold a new component with all required files.

Real-Time Collaboration

Work together with your team in real-time. Multiple cursors, inline comments, and change requests make collaboration seamless.
