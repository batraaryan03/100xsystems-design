Usage Guide

Copy any SVG divider and place it between two sections. Set preserveAspectRatio="none" for responsive stretching. Change the fill color to match your section background. Adjust height for taller/shorter dividers.

<!-- Wave divider --> <svg viewBox="0 0 1440 120" preserveAspectRatio="none" style="display:block;width:100%;height:80px;"> <path d="M0,60 C240,120 480,0 720,60 C960,120 1200,0 1440,60 L1440,120 L0,120 Z" fill="#eef2ff" /> </svg> <!-- Angle divider --> <svg viewBox="0 0 1440 120" preserveAspectRatio="none" style="display:block;width:100%;height:80px;"> <polygon points="0,120 1440,0 1440,120 0,120" fill="#1e293b" /> </svg>
