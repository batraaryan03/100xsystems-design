# Abstract Video Loops

Short, reusable video clips for design projects. These abstract backgrounds and transitions are perfect for hero sections, loading screens, and creative presentations.

## Included Files

- `gradient-flow.mp4` — Slow-moving gradient mesh (10s loop)
- `particle-drift.mp4` — Floating particles on dark background (8s loop)
- `wave-motion.mp4` — Abstract wave animation (12s loop)

## Usage

```html
<video autoplay muted loop playsinline>
  <source src="gradient-flow.mp4" type="video/mp4">
</video>
```
